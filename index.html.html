Introduction to CSS

CSS (Cascading Style Sheets) is a programming language that is used to style online content.
CSS fundamentals explains everything you'll need to get started.

What is CSS?

CSS is not a programming language like HTML.
It's also not a markup language. CSS stands for Cascading Style Sheets.
CSS is used to style HTML elements selectively. This CSS, for example, selects paragraph text and changes the font style.
In order for CSS to adequately apply to an HTML content it has to be linked using the link tag

Anatomy of a CSS Element

A ruleset is the name given to the entire structure.


The main parts of the CSS element are as follows

  • Selector - This is the HTML element name at the start of the ruleset.
    It defines the element(s) to be styled (in this example, <p> elements). To style a different element, change the selector.
  • Declaration - This is a single rule like color: red;. It specifies which of the element's properties you want to style.
  • Properties - An HTML element can be styled in a variety of ways.
    (Color is a property of the <p> elements in this example.)
    In CSS, you select the attributes you want the rule to influence.
  • Property Value - The property value is listed to the right of the property, following the colon.
    This selects one of the many potential appearances for a given property from a large number of options.
    (There are numerous color values other than red, for example.)

Summary

The code that styles online content is called CSS (Cascading Style Sheets).
CSS is not a programming language in the same way that HTML is not.
Not only is it not a markup language, but it is also not a programming language.

CSS (Cascading Style Sheets) is a programming language that allows you to create style sheets.
CSS is the method for styling HTML elements selectively.
Selector, Declaration, Properties, and Property Value are all part of a ruleset.
